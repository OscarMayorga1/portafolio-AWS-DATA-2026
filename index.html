Mi Portafolio de Proyectos curso AWS DATA 2026

Colección de prácticas y proyectos desarrollados en AWS con uso de Python y PySpark.

Proyectos

  • Entregable 1 - Clasificación de Ventas

    Programa en Python para clasificar ventas utilizando estructuras condicionales.

  • Entregable 2 - Fundamentos de datos con CSV, NumPy y Pandas

    Se realizan operaciones básicas sobre datos extraídos de un archivo CSV y convertidos en un DataFrame. A partir de esta información, se aplican diferentes análisis y transformaciones para obtener insights valiosos y facilitar la interpretación de los datos.

  • Entregable 3 - Análisis narrativo del archivo Netflix_titles: datos, limpieza y vizualizaciones

    Se realizó un análisis de datos que incluyó limpieza, creación de variables, exploración de patrones y visualización.

  • Entregable 3.1 - Analisis Netflix_titles:Imagenes de Funcionamiento Streamlite

    Imagenes de funcionamiento de pagina generada con la libreria Streamlite.

  • Entregable 4 - Uso de Boto3 en aws para cargar datos de Titanic

    Se utilizara la libreria Boto3 para utilizar un archivo csv desde un S3.

  • Entregable 5 - Funcion Lambda AWS

    Se creara una funcion lambda que ejecute un ETL cargando un archivo, procesando una limpieza de este y devolviendo el archivo limpio para su analisis, uso de Buckets.

  • Proyecto Integrador - Analisis a Taxes de NuevaYork

    Se realizó un análisis de datos que incluyó limpieza, creación de variables, exploración de patrones y visualización, culminando con un modelo de regresión logística para estimar probabilidades y entender relaciones entre variables.
