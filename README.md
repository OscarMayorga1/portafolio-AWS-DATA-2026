# 📊 Portafolio AWS DATA 2026 — Oscar Mayorga

Colección de prácticas y proyectos desarrollados durante el curso **AWS DATA 2026**, enfocados en análisis de datos, ingeniería de datos en la nube y automatización con servicios de AWS.

🔗 **Ver portafolio en vivo:** [oscarmayorga1.github.io/portafolio-AWS-DATA-2026](https://oscarmayorga1.github.io/portafolio-AWS-DATA-2026/)

---

## 🛠️ Stack técnico

`Python` · `Pandas` · `NumPy` · `PySpark` · `Boto3` · `AWS Lambda` · `Amazon S3` · `Streamlit` · `HTML/CSS`

---

## 📁 Proyectos

| Proyecto | Descripción | Tecnologías |
|---|---|---|
| [Entregable 1 — Clasificación de Ventas](Actividad_1.html) | Programa en Python que clasifica ventas utilizando estructuras condicionales. | Python |
| [Entregable 2 — Fundamentos de datos con CSV, NumPy y Pandas](Actividad_2.html) | Operaciones sobre datos extraídos de un CSV, transformados en un DataFrame para generar insights. | Python, Pandas, NumPy |
| [Entregable 3 — Análisis narrativo de Netflix](Analisis_Datos_Netflix.html) | Limpieza, creación de variables, exploración de patrones y visualización de datos de `netflix_titles`. | Python, Pandas |
| [Entregable 3.1 — Netflix con Streamlit](Analisis_Datos_Netflix_imagenes.html) | Capturas de funcionamiento de la app interactiva construida con Streamlit. | Streamlit |
| [Entregable 4 — Boto3 + Titanic](Boto3_Titanic.html) | Carga de un CSV desde S3 usando Boto3; carrusel de imágenes del proceso. | Boto3, S3 |
| [Entregable 5 — ETL con Lambda](ETL_Lambda_Boto3_Streamlite.html) | Función Lambda que descarga un CSV desde S3, aplica limpieza y guarda el resultado; se invoca desde una app de Streamlit. | AWS Lambda, Boto3, S3, Streamlit |
| [Proyecto Integrador — Taxis de Nueva York](Analisis_Datos_Netflix.html) | Pipeline ETL de datos con PySpark
Pipeline ETL completo: extracción de datos desde una fuente original, ingesta en Amazon S3, limpieza y generación de KPIs con PySpark, y consumo final de los resultados en una app de Streamlit con visualizaciones interactivas. | Python, PySpark, Amazon S3, Streamlit |
